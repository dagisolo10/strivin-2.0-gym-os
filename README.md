# Strivin 2.0 — Gym OS

A modern, offline-first workout tracking system focused on **consistency, progression, and simplicity**.

## 🚀 Overview

Strivin 2.0 helps you:

- Track workouts effortlessly
- Build streaks with “Perfect Days”
- Automatically improve your performance over time

No spreadsheets. No manual calculations. Just train.

## 🧠 Core Features

### 🏁 Smart Onboarding

- Define workout days and splits
- Add exercises with sets, reps, and weights
- Customize your plan anytime

### 📆 Today’s Workout

- Clean daily view
- Fast logging (one tap per set)
- Progress bar for completion

### 🔥 Perfect Day System

- Complete your workout → earn a Perfect Day
- Build streaks
- Stay consistent

### 📊 Progress Tracking

- Weekly workout carousel
- Exercise history
- Personal best tracking

### ⚡ Adaptive Progression

- Automatically adjusts:
  - Weight
  - Reps
- Based on your actual performance

### 🎯 Focus Mode

- Distraction-free workout experience
- Only current exercise visible

### 🏆 Gamification

- Streaks
- Achievements
- Visual progress feedback

### 🔔 Smart Notifications

- Pre-workout reminders
- Post-workout check-ins

### 📈 Analytics & Insights

- Weekly volume tracking
- Strength progression
- Streak calendar

### 📡 Offline First

- Works without internet
- Local-first data storage
- Sync optional

## 🛠 Tech Stack

- React Native (Expo)
- NativeWind (Tailwind for RN)
- Drizzle ORM
- SQLite / PostgreSQL

## ⚡ Setup

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```
